# chat-app
简易的聊天app，基于vue2.0，mongoose，node，socket.io，muse-ui等技术。

# Build Setup

#install dependencies<br>
npm install

#serve with hot reload at localhost:8080<br>
npm run dev

#build for production with minification<br>
npm run build

# Main Technology
客户端：vue2.0、vuex
服务端：nodejs、express
数据库：mongodb
通讯：socket.io
ui框架：muse-ui
